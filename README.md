## About

Cairo-svg is a SVG image library written in Cairo 0.10.

## Getting Started

### 📦 Prerequisites

- [protostar](https://github.com/software-mansion/protostar)

### 🎉 Install

```bash
make install
```

### ⛏️ Compile

```bash
make
```

### 🌡️ Test

```bash
make test
```
